iPhone setup
Install CURB for alerts

Apple only allows push notifications once CURB lives on your Home Screen as an app.

  1. 1Tap the Share button in Safari's toolbar
  2. 2Choose Add to Home Screen
  3. 3Open CURB from its new icon, then tap 🔔 Sweep alerts

Already a Home Screen app? Make sure you opened it from the icon (not Safari), then try again.
